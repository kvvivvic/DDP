# DDP

---

### [https://kvvivvic.github.io/DDP/](https://kvvivvic.github.io/DDP/)

## 프로젝트 보고서

---

### 사용 Skill

- HTML
- CSS
- JavaScript
- JQuery

### 사이트 설명

- 반응형 하이브리드 웹&앱(PWA) 로 만들어진 사이트입니다.
- PWA로 만들었기떄문에 앱으로 다운받아 사용할 수 있습니다.
- 기존에 있는 DDP 사이트를 리뉴얼 하였습니다.
- Page 별 다양한 인터렉션이 구현되어 있습니다.

### 분기점

- 1920~1200px (PC)
- 1200~1024px (태블릿)
- 1024~768px (모바일)
